Imputing missing values

We have noticed that there appears to be missing values in the data set. The first thing is to count the number of missing values. Then we will use the daily max to imput those values.

missing <- sum(is.na(dat[,1]))

There are 2304 values in the data set. Now the missing values will be imputed with the daily averages and a histogram will be generated.

timestamp <- dat[c(3,4,5)]

merge <- merge(avg.steps, timestamp, by="interval")

dat.mod <- merge(dat, merge, by="datetimestamp")

for (i in 1:length(dat[,2])) {
        if (is.na(dat[i,2])) {
                dat[i,2] <- dat[i,10]
        }
}

total.steps <- as.data.frame(aggregate
